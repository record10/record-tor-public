Installation Guide

How to install on each platform

Quick commands and steps to get RecordTor running on your system.

laptop_mac macOS

  1. Download the .dmg file
  2. Open the .dmg and drag RecordTor to Applications
  3. After installing, run in Terminal:
    xattr -cr /Applications/RecordTor.app
    Then open the app normally

window Windows (Setup)

  1. Download the -setup.exe file
  2. Run the installer and follow the prompts
  3. If SmartScreen warns about an unknown publisher, click More info → Run anyway

window Windows (MSI)

  1. Download the .msi file
  2. Double-click to run the MSI installer
  3. If SmartScreen warns about an unknown publisher, click More info → Run anyway

android Android

  1. Download the .apk file to your device
  2. Open the file & allow installation from unknown sources when prompted
  3. Tap Install and then Open

terminal Linux (AppImage)

  1. Download the .AppImage file
  2. Make it executable:  chmod +x RecordTor*.AppImage
  3. Run it:  ./RecordTor*.AppImage

terminal Linux (Debian)

  1. Download the .deb file
  2. Install via terminal:  sudo dpkg -i recordtor*.deb
  3. Fix dependencies if needed:  sudo apt-get install -f

shield About Unsigned App Warnings

RecordTor is not code-signed with a paid certificate, so your OS may show a security warning on first launch. This is normal for open-source software distributed outside app stores. The app is safe to use.

  • macOS: After installing, open Terminal and run: xattr -cr /Applications/RecordTor.app — this removes the quarantine flag so the app can launch normally. You only need to do this once.
  • Windows: When SmartScreen appears, click More info → Run anyway.
  • Android: Enable Install from unknown sources for your browser or file manager in Settings → Security.
